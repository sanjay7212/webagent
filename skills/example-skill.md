# Example Skill

This is a sample skill file. Create new `.md` files in this directory to add skills.

Each skill file should have YAML frontmatter with:
- **name**: A short identifier for the skill
- **description**: What this skill does
- **trigger**: When this skill should be activated

The body contains the instructions the agent should follow when this skill is triggered.
